## 🚀 Tentang Project
**SpendWise** adalah platform dompet digital modern yang memfasilitasi transaksi pembayaran tagihan (seperti BPJS) dan *Top Up E-Wallet* dengan antarmuka *Dark Glassmorphism* yang elegan, aman, dan responsif.

## ✨ Fitur Utama
*   💸 **Top Up Cepat:** Dukungan ke berbagai provider (*Gopay, DANA, ShopeePay*).
*   🏥 **Pembayaran BPJS:** Cek dan bayar tagihan BPJS Kesehatan dalam hitungan detik.
*   🎨 **Premium UI/UX:** Dibangun dengan filosofi desain *Glassmorphism*.
*   🔒 **Keamanan Berlapis:** Validasi PIN 6-digit untuk setiap transaksi.
